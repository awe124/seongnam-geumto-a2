# 성남금토 A-2블록 국민임대주택 랜딩페이지

GitHub Pages에 바로 올릴 수 있는 정적 랜딩페이지입니다.

## 파일 구성

- `index.html` : 메인 랜딩페이지
- `style.css` : 반응형 스타일
- `assets/landing.jpg` : 랜딩페이지 이미지

## GitHub Pages 배포

1. GitHub에서 새 repository 생성
2. 이 폴더 안의 파일 전체 업로드
3. Repository 상단 `Settings` 이동
4. 좌측 `Pages` 선택
5. `Build and deployment`에서 Source를 `Deploy from a branch`로 선택
6. Branch를 `main` / Folder를 `/root`로 설정 후 Save
7. 잠시 후 생성되는 Pages URL로 접속
